# 7TV-Emotes
Search emotes from 7TV (based on [Garulf's BetterTTV-Twitch-Emotes](https://github.com/Garulf/BetterTTV-Twitch-Emotes))

# Installation:
Now available in Flow Launcher Plugin Store (v0.0.3)

Flow Launcher:
```
pm install 7TV Emotes
```
